# Release Notes - v5.0.0 Repo Polish Package

This package upgrades the World Cup 2026 prediction repository from a working handoff into a cleaner GitHub publishing bundle.

## Highlights
- Added `LICENSE` with MIT terms
- Added `CHANGELOG.md`
- Added `CONTRIBUTING.md`
- Added GitHub Actions CI workflow
- Added issue templates and pull request template
- Expanded README for GitHub publishing, setup, and deployment
- Preserved the Streamlit dashboard, trained outputs, and source-ready files

## Best use cases for this package
- publishing the project to GitHub
- handing the repo to collaborators
- deploying the dashboard to Streamlit Community Cloud
- using the repo as a base for Version 5 product features such as backtesting and scenario simulation
